# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-05-10

### Added
- Initial release of Fish CLI
- `fish generate` — Generate audio from text using a cloned voice (TTS)
- `fish clone-voice` — Clone a voice from an audio URL
- `fish task` — Query a single task status
- `fish tasks` — Query multiple tasks at once
- `fish wait` — Poll until a task completes
- `fish models` — List available Fish models
- `fish config` — Show current configuration
- `--json` flag on all commands for machine-readable output

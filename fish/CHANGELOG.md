# Changelog

## [0.1.0] - 2026-06-30

### Added

- Initial release of Fish CLI
- `tts` command for text-to-speech generation
- `models` command to list available voice models
- `model` command to get a specific voice model
- `task` command to query a single task
- `tasks` command to query multiple tasks
- `wait` command to poll until task completion
- `tts-models` command to list available TTS model versions
- `config` command to show current configuration

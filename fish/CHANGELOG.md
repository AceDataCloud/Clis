# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Calendar Versioning](https://calver.org/) (YYYY.MM.DD.BUILD).

## [0.1.0] - 2026-05-10

### Added
- Initial release
- Fish AI text-to-speech via `fish tts`
- Voice model listing via `fish voices` and `fish voice <id>`
- Task management with polling (`fish task`, `fish tasks`, `fish wait`)
- Rich terminal output with tables and panels
- JSON output mode (`--json` flag)
- Multiple TTS model support (s1, s2-pro)

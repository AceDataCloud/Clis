# Changelog

## [0.1.0] - 2025-01-01

### Added
- Initial release
- `chat` command using OpenAI-compatible endpoint (`/gemini/chat/completions`)
- `generate` command using native Gemini API (`/v1beta/models/{model}:generateContent`)
- `models` command to list available models
- `config` command to show current configuration
- Support for models: gemini-2.0-flash, gemini-2.5-flash, gemini-2.5-pro, gemini-3-flash-preview, gemini-3.0-pro, gemini-3.1-pro

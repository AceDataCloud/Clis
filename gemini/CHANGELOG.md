# Changelog

## [0.1.0] - 2025-05-20

### Added
- Initial release of Gemini CLI
- `chat` command for Gemini chat completions (`/gemini/chat/completions`)
- `models` command to list available Gemini models
- `config` command to show current configuration
- Support for models: gemini-3.1-pro, gemini-3.0-pro, gemini-3.5-flash, gemini-3-flash-preview, gemini-2.5-pro, gemini-2.5-flash, gemini-2.0-flash

# Changelog

## [0.1.0] - 2025-06-25

### Added
- Initial release of Gemini CLI
- `chat` command for Gemini chat completions (`/gemini/chat/completions`)
- `generate` command for text-to-video generation (`/gemini/videos`)
- `image-to-video` command for image-to-video generation (`/gemini/videos`)
- `task` command to query a single task status
- `tasks` command to query multiple tasks
- `wait` command to poll until a task completes
- `models` command to list available Gemini models
- `config` command to show current configuration
- Support for chat models: gemini-3.1-pro, gemini-3.0-pro, gemini-3.5-flash, gemini-3-flash-preview, gemini-2.5-pro, gemini-2.5-flash, gemini-2.0-flash
- Support for video model: omni-flash

# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2024-01-01

### Added
- Initial release
- `imagine` command for image generation
- `edits` command for image editing
- `videos` command for video generation
- `describe` command for image description
- `shorten` command for prompt shortening
- `translate` command for content translation
- `seed` command for retrieving image seeds
- `task` and `tasks` commands for task management
- `wait` command for polling task completion

## [0.1.0] - 2026-07-09

### Added
- Initial release
- HappyHorse video generation from text prompts (generate)
- Image-to-video generation (image-to-video)
- Reference-to-video generation (reference-to-video)
- Video editing (video-edit)
- Task management with polling
- Rich terminal output with tables and panels
- JSON output mode (`--json` flag)
- Multiple model support

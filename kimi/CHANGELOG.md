# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-07-18

### Added
- Initial release of Kimi CLI
- `kimi chat` — Chat with a Kimi model (chat completions)
- `kimi models` — List available Kimi models
- `kimi config` — Show current configuration
- `--json` flag on all commands for machine-readable output

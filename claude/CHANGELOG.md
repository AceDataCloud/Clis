# Changelog

## [0.1.0] - 2025-01-01

### Added
- Initial release
- `chat` command using OpenAI-compatible endpoint (`/v1/chat/completions`)
- `messages` command using native Claude Messages API (`/v1/messages`)
- `count-tokens` command to count tokens in messages (`/v1/messages/count_tokens`)
- `models` command to list available models
- `config` command to show current configuration
- Support for 16 Claude models

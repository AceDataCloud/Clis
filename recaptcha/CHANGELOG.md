# Changelog

## [0.1.0] - 2024-01-01

### Added
- Initial release
- `recognize` command for reCAPTCHA v2 image recognition
- `token` command for reCAPTCHA v2 token generation
- `token3` command for reCAPTCHA v3 token generation
- `config` command to show current configuration
- Support for async mode via `--async` flag
